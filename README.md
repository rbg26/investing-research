# investing-research

A Claude Code plugin that turns Claude into an end-to-end investment research analyst. Give it a stock ticker and it produces a full research package: Porter's Five Forces industry analysis, 3-year financial statement analysis, a 5-tab Excel valuation model, and a Munger/Buffett-style investment checklist.

## What it produces

| Output | Format | What's inside |
|--------|--------|---------------|
| Industry & strategy analysis | PDF | Porter's Five Forces, strategy coherence, competitive advantage, failure modes |
| Financial statement analysis | HTML dashboard | 3-year IS/BS/CF, 20+ ratios with formulas, DuPont decomposition, analyst synthesis |
| Valuation model | Excel (.xlsx) | Comps, DCF (Bear/Base/Bull), Sensitivity, Conservative DCF, Dhandho DCF |
| Investment checklist | PDF | 35-question Munger/Buffett checklist, scorecard, analyst synthesis, "What Would Munger Ask?" section |

## Commands

### `/deep-dive [TICKER]`
Runs all four stages sequentially and delivers every output in a single session. This is the main command — use it when you want the full research package.

### `/full-valuation [TICKER]`
Runs only the valuation model (Stage 3). Use this when you already have the research context and just need the Excel model.

## Requirements

### MCP servers (required)
This plugin pulls all financial data through two MCP servers. You must have both configured.

**edgartools** — SEC EDGAR filings (10-Ks, financial statements)
```json
{
  "edgartools": {
    "type": "stdio",
    "command": "uvx",
    "args": ["--from", "edgartools[ai]", "edgartools-mcp"],
    "env": {
      "EDGAR_IDENTITY": "Your Name your@email.com"
    }
  }
}
```
Set `EDGAR_IDENTITY` to your name and email. The SEC requires this as a User-Agent string for API access.

**yfinance** — market data, current prices, ratios
```json
{
  "yfinance": {
    "type": "stdio",
    "command": "uvx",
    "args": ["yfmcp@latest"]
  }
}
```

### Python (required for valuation model)
The valuation model runs a Python script that builds the Excel file. You need Python 3 and `openpyxl`:
```bash
pip3 install openpyxl
```

### Public skills (resolved automatically)
The skills reference two Anthropic-hosted public skills for PDF and HTML output generation. These are resolved automatically — no setup needed.

## Installation

```
/plugins add github:YOUR_GITHUB_USERNAME/investing-research-plugin
```

Then install the plugin:
```
/plugins install investing-research
```

## How it works

The plugin has four skills that can be run independently or chained together by the `deep-dive` command:

1. **`strategy-industry-analysis`** — Fetches the 10-K from EDGAR, runs 15–25 web searches, and writes a full Porter's Five Forces + strategy analysis PDF. Includes inversion (failure modes), circle of competence disclosure, and self-scoring against a 5-dimension rubric.

2. **`financial-statement-analysis`** — Pulls 3 years of financial data, calculates 20+ ratios (with formulas displayed), runs a full earnings quality bridge (OCF vs net income), DuPont decomposition, and produces an HTML dashboard with analyst commentary.

3. **`full-valuation-model`** — Pulls data from yfinance and EDGAR, selects 4–6 comparable peers, sets Bear/Base/Bull assumptions anchored to historical data, and runs a Python script to build a 5-tab Excel workbook with live formulas throughout.

4. **`investment-checklist`** — Runs a 35-question Munger/Buffett checklist using Claude with extended thinking, synthesises findings from the prior three stages, and produces a PDF with verdict badges, a scorecard, an analyst synthesis, and a "What Would Munger Ask?" section.

## Notes

- The `deep-dive` command does not pause between stages — it runs all four sequentially and delivers everything at the end. Expect 10–20 minutes for a full run.
- All valuation model cells are live Excel formulas — nothing is hardcoded. You can change any assumption and the model recalculates.
- The Conservative DCF and Dhandho DCF tabs use value-investing guardrails (end-of-year discounting, 12–15% discount rates, 1% terminal growth) that are deliberately conservative.
- The "What Would Munger Ask?" section in the checklist is generated by Claude Opus with extended thinking, synthesising all four stages of research.
